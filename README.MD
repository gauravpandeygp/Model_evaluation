# Task 3 - Model Evaluation and Prediction

## Description

This project evaluates a Linear Regression model using Mean Absolute Error (MAE) and predicts the exam score for a student who studies 4.5 hours.

## Features

- Creates a sample dataset
- Splits data into training and testing sets
- Trains a Linear Regression model
- Predicts test data
- Calculates Mean Absolute Error (MAE)
- Compares Actual vs Predicted values
- Predicts score for 4.5 study hours

## Technologies Used

- Python
- Pandas
- Scikit-learn

## How to Run

1. Install the dependencies:

```bash
pip install -r requirements.txt
```

2. Run the program:

```bash
python model_evaluation.py
```
